Laravel Project
Requirements

Pastikan perangkat Anda telah terinstall:

PHP >= 8.2
Composer
MySQL / MariaDB / Sqlite
Git

Installation
1. Clone Repository
git clone https://github.com/richardsoegito/test-matahari-sakti.git

Masuk ke folder project.

cd repository
2. Install Dependency PHP
composer install

3. Menjalankan project
php artisan serve

4. Migrasi database tabel
php artisan migrate

5. Seeder
php artisan db:seed
